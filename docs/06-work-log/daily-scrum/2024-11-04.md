- 성공 코드 200 or 201로 통일, 실패 코드는 각자 커스텀

- 엔티티 클래스의 필드명은 도메인명을 붙이지 않기
```java
ex) 
    @Id
    @GeneratedValue(strategy = GenerationType.UUID)
    private UUID id;

    @Column(nullable = false)
    private String name;

    @Column(nullable = false)
    private String content;

    @Column(nullable = false)
    private int price;
```

- 튜터님 피드백:
  - MSA 프로젝트에서는 API Gateway에서 '인증'만, 나머지 서비스에서 '인가'를 하는 게 좋아요
  - 왜냐하면 Token 값은 모든 서비스에서 공통으로 쓸 수 있지만
  - 권한 같은 경우에는 서비스마다 다를 수 있기 때문이에요
